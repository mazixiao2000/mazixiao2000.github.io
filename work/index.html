RACING LEVEL DESIGN / CAMERA SYSTEM

GAME DESIGN / LEVEL DESIGN

FLING

一款以钓竿摆荡与精准操作为核心的 2D 平台跳跃游戏，负责核心玩法设计与大部分关卡设计，通过重新定义核心机制挽救 Prototype，并完成从教学到高难挑战的完整难度曲线设计。

2D横板平台跳跃类《蔚蓝》
游戏设计 / 关卡设计核心玩法设计 · 大部分关卡设计 · 难度曲线
